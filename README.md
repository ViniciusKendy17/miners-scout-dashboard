
# Miners Scouting Dashboard

I also developed the app for the data collection, but unfortunately I cannot provide the code.

---  

The FRC Scouting Dashboard is a web-based platform designed to connect with a mobile scouting application.

The mobile app is used during FRC competitions to collect real-time data about robots. All collected data is sent to this dashboard, where it is processed and analyzed to produce reports, synergy calculations, and rankings.

These insights help teams make strategic decisions and alliance selections during competition.

## Field Use

This solution was used by the MinerSkills team (Minas Gerais, Brazil) during the official 2025 Brazilian National FRC Qualifier.
It supported the team throughout the event with reliable data analysis, synergy evaluation, and ranking insights for alliance strategy.

## Key Capabilities

Connects with the custom mobile scouting app

Collects and processes match data from competitions

Generates performance reports, synergy data, and rankings

Supports data-driven decisions during scouting and alliance selection

**Data-driven insights for smarter competition strategies.**
